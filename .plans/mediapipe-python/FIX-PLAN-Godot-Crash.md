## Issues Identified

### Issue 1: Missing Main Scene Recognition (CRITICAL)
**Error:** `Error: Can't run project: no main scene defined in the project.`  
**Location:** `~/.local/share/godot/app_userdata/[unnamed project]/logs/`  
**Root Cause:** Project opens as "[unnamed project]" instead of "AeroBeat Assembly"

**Fix Required:**
- Verify `project.godot` has correct `config/name`
- Verify `run/main_scene="res://scenes/main.tscn"` is set
- Clear `.godot/` cache folder to force re-import
- Ensure `main.tscn` exists and is valid
