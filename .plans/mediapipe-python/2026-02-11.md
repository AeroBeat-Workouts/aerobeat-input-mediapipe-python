## Notes

- **Godot Caching Issue:** Editor sometimes hangs when files edited externally. Workaround: close Godot before agent edits, delete `.godot/` folder if needed.

- **Known Godot Behavior:** `StreamPeerTCP` status transitions can be tricky. Must poll frequently and check status after each operation.

- **MJPEG Format:** Server sends `multipart/x-mixed-replace` with boundaries. JPEG data is binary, headers are ASCII.
