## Technical Architecture

### Components

| Component | Status | Location |
|-----------|--------|----------|
| AutoStartManager | ✅ Works | `src/autostart_manager.gd` |
| MediaPipeServer | ✅ Works | `src/server/mediapipe_server.gd` |

| Test scene | 🔄 Hanging | `test/test_scene.tscn` |

### Communication Flow

```
[Godot Test Scene]
    ↓ (AutoStartManager spawns)
[Python main.py] ←── venv + PYTHONPATH
    ↓
[MediaPipe Pose Detection]
    ↓
UDP:127.0.0.1:4242 ←── JSON landmarks
    ↓
[mediapipe_server.gd]
    ↓
[Game Logic]
```

### Key Fixes Applied

1. **Detached process spawning** - Uses bash with output redirected to prevent pipe blocking
2. **DISPLAY auto-detection** - Tries `:1`, `:0`, `:2` for Linux X11
3. **PID detection via pgrep** - Finds actual Python process, not bash wrapper
