Truth guard: skip_sidecar_stop_on_close_debug is the broad override. If it is on, the effective shutdown mode is always heartbeat_only and the narrower Inspector toggles do not matter. Also note that boxing_proving.tscn currently bakes that broad override on by default, so narrow-mode testing is misleading unless you first change the root Inspector values.
